# 个人预算管理工具（Budget Tracker）

欢迎使用 **个人预算管理工具**！  
这是一个简单易用的命令行程序，帮助你记录收入和支出、管理个人预算，并支持文件保存与加载。

---

## ✨ 功能特点

### 1. 添加一笔记录
支持添加 **收入** 或 **支出**，并填写：
- 金额
- 类别
- 备注
- 日期（自动生成）

### 2. 查看所有记录
以清晰的列表格式展示你所有的历史记录。

### 3. 查看统计信息
自动计算：
- **总收入**
- **总支出**
- **当前余额**

### 4. 保存记录到文件
将所有预算记录保存到本地文件（如 JSON 或文本格式），方便下次继续使用。

### 5. 从文件加载记录
读取先前保存的文件，恢复你的预算记录。

### 0. 退出程序
安全退出应用。
